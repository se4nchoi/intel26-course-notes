# Intel-26 K-DT Course Notes

Notes, practice code, and technical artifacts from the Intel-26 K-DT training program.

This repository is organized by major course area. It includes raw class notes, reviewed summaries, exercises, datasets, and software-specific project files.

## Course Areas

### [Python Programming](python_programming/)

Python fundamentals, object-oriented programming, NumPy, Pandas, data analysis, introductory machine learning, and coding-test practice.

### [PLC and Automation](plc/)

Mitsubishi MELSEC-Q PLC programming using GX Works2, including ladder logic, sequence control, I/O mapping, HMI screens, and automation exercises.

Additional course areas may be added as the program progresses.
