# Intel-26 K-DT Course Notes

Daily notes, practice code, and review material from the Intel-26 K-DT training program.

I use this repository to organize what I learn during class, rewrite key topics in my own words, and keep small examples that I can revisit later.

## Current Topics

* Python fundamentals
* Object-oriented programming
* NumPy
* Pandas
* Data analysis
* Introductory machine learning
* Regression and classification

The list will be updated as major parts of the program are completed.

## Repository Structure

Notes are mainly organized by month and class date.

```text
July/
├── 0701/
├── 0702/
├── 0707/
└── ...

Each date may contain:
├── notes.md
├── practice code
├── datasets
└── generated outputs
```

Some folders contain raw lecture notes, while others include cleaned summaries or additional review work.

## How I Use This Repository

My general process is:

1. Record notes and code during class
2. Review the material after the lecture
3. Rewrite important ideas in a shorter summary
4. Add examples when a concept is unclear
5. Commit the result as a daily learning record

The notes are written mainly for my own review, so some files may be incomplete or informal.

## Notes on AI Use

AI tools are sometimes used to reorganize raw lecture notes, explain unfamiliar concepts, generate review questions, and improve commit messages.

I keep my own observations and notes where they help show how my understanding developed.

## Status

This is an active learning repository rather than a finished course guide.

Material may include incomplete exercises, experimental code, duplicated examples, or notes that have not yet been fully reviewed.
