====	Labeled Program	=====
* structured program = very complicated systems (not for training)

* simple program -> checked with label

- label = variable names (how will you tell what T123 was made for ?)

labeled internal functions (eg. timer, counter) will default to assigning from the LAST available numbering 
(i.e a labeled local counter will start at C2047 since 2K counters are avail)

VAR, VAR_CONSTANT, VAR_INPUT, VAR_OUTPUT, VAR_RETAIN
