# PLC Course Notes

PLC training materials from the  Intel-26 K-DT training program, focused on Mitsubishi MELSEC-Q PLCs using the Q02H CPU.

## Contents

- `.gxw` — GX Works2 ladder logic projects
- `.prx` — HMI screen and project files
- `.txt` / `.md` — course notes and explanations
- `.xlsx` — PLC I/O mappings, reference tables, and bit/hex calculators

## Topics

- PLC components, I/O modules, sensors, relays, motors, and solenoid valves
- Ladder logic, self-holding circuits, interlocks, sequence control, and `SET` / `RST`
- Boolean logic, truth tables, De Morgan’s theorem, and Karnaugh maps
- Timers, counters, data registers, and indirect setpoints
- Binary, hexadecimal, bitmasks, and 16-bit word operations
- Data transfer, comparison, arithmetic, shift, and pulse instructions
- Conveyor, pneumatic, analog I/O, encoder, positioning, and servo control
- HMI, serial, Ethernet, CC-Link, and MELSECNET communication
